**New Features**

**Cost Analytics Dashboard** — Track and visualize LLM usage costs with detailed breakdowns, histograms, and filtering options on Self-Hosted.

**Image Compression** — Configurable image compression for attachments to optimize storage and processing.

**Fuzzy Search** — Database-backed fuzzy search for semantic search with improved partial matching. [Learn more →](/api-reference/endpoints/semantic-search)

**Improvements**

Improved analytics date range selection across all widgets. Enhanced prompt templates for better context understanding. Faster label selection in semantic search (from ~15s to ~1.3s). Better log chunk selection with stratified sampling.

</Update>
