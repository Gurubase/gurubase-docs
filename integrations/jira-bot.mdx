## Introduction

Integrate Gurubase with Jira to use your Jira issues as data sources for Gurubase.


## Steps to Integrate

1. **Create your API Key**
    - Go to your [Jira API key settings](https://id.atlassian.com/manage-profile/security/api-tokens)
    - Click "Create Classic API token"
    - Copy it

2. **Navigate to Integrations**
   - Go to the Gurubase dashboard
   - Click on "My Gurus" under the profile
   - Click on the Guru you want to integrate with Jira
   - Click on the "Jira Bot" option

3. **Fill in the fields**
    - Domain
    - User Email
    - API Key

<Frame>
  <img src="/images/jira-fields.png" alt="Jira Integration Fields" />
</Frame>

## Using the Bot

In the Guru edit page, you can now use the "Add Jira Issues" button to add them as data sources.
