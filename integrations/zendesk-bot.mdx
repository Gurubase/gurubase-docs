## Introduction

Integrate Gurubase with Zendesk to use your Zendesk tickets/articles as data sources for Gurubase.


## Steps to Integrate

1. **Create your API Key**
    - Go to your Zendesk API key settings. Its URL is of format:

    `https://{subdomain}.zendesk.com/admin/apps-integrations/apis/zendesk-api/settings/tokens`
    
    Replace `{subdomain}` with your subdomain.
    - Click "Add API token"
    - Copy it

2. **Navigate to Integrations**
   - Go to the Gurubase dashboard
   - Click on "My Gurus" under the profile
   - Click on the Guru you want to integrate with Zendesk
   - Click on the "Zendesk Bot" option

3. **Fill in the fields**
    - Domain
    - User Email
    - API Key

## Using the Bot

In the Guru edit page, you can now use the "Add Zendesk Data" button to add your tickets and articles as data sources.
